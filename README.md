# MI Boy Color - Shell

Open-source **handheld shell** designed in [DoodleDev](https://doodledev.app).

This repository is the shell only: HTML/CSS device chrome and face assets.  
It does **not** include the emulator, menus, Controls app, or game library.

## Play the real product

## Run the shell demo

```bash
npx serve .
```

You should see the handheld with a “SHELL DEMO” LCD card. Use **Play MI Boy →** for the full experience.

## What’s included

- Inlined DoodleDev device markup
- Shell stylesheet (`css/gameboy-bundle.css`)
- Face art: `base.webp`, A/B buttons, favicon / OG image

## What’s not included (private product)

- Game Boy emulator core
- On-device menus, audio, Controls rebinding
- ROM library / upload
